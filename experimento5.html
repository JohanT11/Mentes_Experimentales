Fuente de Poder Casera

Una fuente de poder casera es un proyecto útil para aprender sobre circuitos eléctricos y la regulación de voltaje. Este experimento te enseñará a construir una fuente de alimentación básica que puede proporcionar diferentes voltajes para tus proyectos electrónicos.

Materiales

  • Transformador de 12V
  • Puente rectificador
  • Capacitores (1000µF y 220µF)
  • Regulador de voltaje (LM317 o similar)
  • Resistencias
  • Potenciómetro
  • Placa de circuito impreso
  • Cables de conexión
  • Soldador y estaño

Paso a Paso

  1. Preparar el Transformador: Conecta el transformador de 12V al puente rectificador para convertir la corriente alterna (AC) a corriente continua (DC).
  2. Conectar los Capacitores: Usa los capacitores para suavizar la salida del puente rectificador y reducir el rizado en la corriente.
  3. Instalar el Regulador de Voltaje: Conecta el regulador de voltaje a los capacitores para ajustar la salida a un voltaje deseado.
  4. Ajustar el Potenciómetro: Usa el potenciómetro para ajustar el voltaje de salida a tu necesidad.
  5. Verificar el Funcionamiento: Usa un multímetro para medir la salida y asegurarte de que la fuente de poder esté funcionando correctamente.
