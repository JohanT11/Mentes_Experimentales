Bobina de Tesla

La bobina de Tesla es un tipo de transformador resonante que se utiliza para generar altos voltajes a frecuencias muy altas. Este experimento es fascinante para estudiar la transferencia de energía inalámbrica y los efectos de las descargas eléctricas de alta frecuencia.

Materiales

  • Bobina primaria (alambre de cobre grueso)
  • Bobina secundaria (alambre de cobre fino)
  • Capacitor de alta tensión
  • Fuente de alimentación
  • Transformador de encendido (Spark Gap)
  • Base de soporte
  • Cables y conectores
  • Herramientas de soldadura

Paso a Paso

  1. Construir la Bobina Secundaria: Enrolla el alambre de cobre fino alrededor de un tubo cilíndrico para formar la bobina secundaria.
  2. Construir la Bobina Primaria: Enrolla el alambre de cobre grueso en forma de espiral para formar la bobina primaria.
  3. Montar el Capacitor: Conecta el capacitor de alta tensión al sistema para almacenar energía.
  4. Instalar el Transformador: Coloca el transformador de encendido (spark gap) para generar descargas eléctricas.
  5. Conectar el Sistema: Conecta todos los componentes según el esquema de conexión para que la bobina de Tesla funcione correctamente.
